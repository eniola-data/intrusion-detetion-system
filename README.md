# Intrusion Detection System

This project uses a deep learning CNN-LSTM model to detect cyber attacks in network traffic.

## Key Details
- Dataset: UNSW-NB15
- Model: CNN + LSTM
- Accuracy: ~98%
- Tools: Python, TensorFlow, Scikit-learn

## Description
This project was developed as part of my MSc in Artificial Intelligence and Data Science. It demonstrates the use of deep learning for cybersecurity intrusion detection.
